# Currency Converter

## Автор
Тамерлан

## Как получить API-ключ
1. Зарегистрироваться на exchangerate-api.com.
2. Скопировать свой ключ.
3. Вставить ключ в переменную API_KEY в коде.

## Примеры использования
1. Запустить приложение: python main.py
2. Выбрать валюты, ввести сумму, нажать "Конвертировать".
3. История сохраняется в history.json.

## Возможные тесты
- Ввести отрицательную сумму — должна быть ошибка.
- Оставить поле суммы пустым — ошибка.
- Проверить, что история сохраняется и загружается.
