# Currency Converter

## Автор
Тамерлан

## Как получить API-ключ
1. Зарегистрироваться на exchangerate-api.com.
2. Скопировать свой ключ.
3. Вставить ключ в переменную API_KEY в файле main.py.

## Запуск приложения
1. Установите зависимости: pip install requests
2. Запустите: python main.py

## Примеры использования
- Введите сумму и выберите валюты.
- Нажмите "Конвертировать".
- История сохраняется в history.json.
